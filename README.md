# CollegeCompanion
The application “College Companion” describes the development of a real – time Android
based mobile application. The application is developed to provide students the
information regarding college activities such as general notices, examination schedules,
attendance, faculty details, syllabus etc. Usually we get this information from notice
board which is the only source of communication between college and its students and
is not much effective. But with this application the idea is to digitize the information and provide it to the students in a much effective way!!
